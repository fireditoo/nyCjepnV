# 项目简介

## 核心功能
- 基于 YOLOv8 的裂缝检测系统，支持单张图片、图片文件夹、视频文件及本地摄像头的检测
- 提供完整源码、标注数据集（2400 张裂缝图片）、训练好的模型及训练结果
- 支持批量检测，界面实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度、IOU阈值，可选择显示标签和原图
- 提供多种检测模型选择，支持检测结果保存
- 包含完整的项目运行教程和 UI 界面源文件

## 适用场景
- 建筑物表面裂缝检测
- 道路桥梁结构监测
- 工业设备表面缺陷检测
- 地质灾害预警分析
- 自动化质量控制系统

## 技术特点
- 基于 Python 3.9 开发，使用 OpenCV、PyQt5、Ultralytics、PyTorch 等技术栈
- 采用 YOLOv8 目标检测算法，具备高精度和实时性
- 提供图形化用户界面，操作简单直观
- 支持多种输入源和输出格式，灵活适应不同应用需求
